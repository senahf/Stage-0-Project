HTML - Hypertext Markup Language

Tags

  • Bold - <b>Content</b>
  • Italics - <em>Content</em>

Computers are Stupid

Programmers say this because computers will interpret the language literally without making inferences.

Missing an End Tag

If you do not close your opening tag with an end tag, the rest of the content following the opening tag will be marked up.

HTML Attributes

  • <tag attr="VALUE">contents</tag>
  • anchor - <a href="http://website.url/">derp</a> ~ Makes "derp" into a link
  • image - <img src="http://link.to/image.png" alt="text shown when image is broken"></img>

Whitespace

  • All whitespaces turn into a single space
  • Use <br> to break into a new line
    • Void Tag - No need to close the tag
  • Use <p>content</p> to create a paragraph text that'll break into a new line

Inline vs Block

  • Inline - Formatting that is in line with the content
    • <br>
    • <img>
    • <em>
  • Block - Formatting that puts the content into its own block
    • <p> - block

HTML Document Structure

  • doctype - What kind of HTML document it is
  • HTML tag - Surrounds the entire HTML Document
  • head tag - metadata, include JS/CSS
    • Title tag - Displays the title on the top of the page
  • body tag - actual contents of the document
